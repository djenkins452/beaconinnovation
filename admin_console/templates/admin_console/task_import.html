Expected JSON Format

Upload a JSON file with an array of tasks or an object with a tasks key:

Array format:
[
  {
    "title": "Task Title",
    "description": {
      "objective": "What to accomplish",
      "inputs": ["Context"],
      "actions": ["Step 1", "Step 2"],
      "output": "Expected result"
    },
    "priority": 2,
    "phase": "Phase 1"
  }
]
Object format:
{
  "tasks": [
    { ... }
  ]
}
Fields:
  • title - Task title (required)
  • description - JSON with objective, inputs, actions, output (required)
  • priority - 1=Low, 2=Medium, 3=High, 4=Critical (optional, default: 2)
  • phase - Phase name (optional)
